**吉伊大师用户使用说明书（测试版）**

呀哈～欢迎来找本鼠鼠。下面这份说明，帮你快速上手、问得更准、避开误区。

**1. 怎么开始用**
1. 打开你拿到的网址，进入登录页。
2. 输入管理员发给你的`账号 + 密码`登录。
3. 登录后进入聊天页，直接在输入框提问。
4. 用完记得点`退出登录`（尤其在公用电脑上）。

**2. 怎么问更有效（重点）**
1. 先说目标：你想解决什么。
2. 再给背景：时间范围、场景、你当前状态。
3. 命理类建议补充资料：
- 姓名
- 出生日期（`YYYY-MM-DD`）
- 出生时间（越准确越好，不知道也可先问）
4. 一次一个核心问题，答案会更聚焦。
5. 遇到听不懂的回答可以复制粘贴继续追问哦～吉伊大师乐意为你解答！

可直接套用：
- `我接下来30天的重点是什么？`
- `今年下半年事业上我该先做哪件事？`
- `我叫XX，2001-08-15，早上7点半出生，想看今年财运节奏。`

**3. 目前支持什么**
1. 账号密码登录后聊天咨询。
2. 运势/节奏/方向类问题（年、月、近30天等）。
3. 星座、解梦、日常情绪陪伴类问答。
4. 连续多轮追问（可基于上下文继续聊）。

**4. 当前局限（请务必看）**
1. 当前是`password-only`模式：
- 不支持短信注册
- 不支持短信登录
- 不支持短信找回密码
2. 新用户需管理员预置账号。
3. 忘记密码走人工流程（客服核验后管理员重置）。
4. 命理结果是参考建议，不替代医疗/法律/投资等专业意见。
5. 若缺少出生时辰，命理细节准确度会下降。
6. 偶发模型波动可能导致表述不稳，建议用“再细化一下”继续追问。

**5. 隐私与安全建议**
1. 不要在聊天里发身份证号、银行卡、验证码等敏感信息。
2. 公共设备使用后务必退出登录。
3. 如发现异常回复或页面问题，截图并反馈给管理员。

**6. 遇到问题怎么办**
1. 登录失败：先确认账号密码，再联系管理员。
2. 回答不够准：补充背景和时间范围，要求“给结论+依据+可执行步骤”。
3. 需要改密/新开号：走管理员人工处理。

要的话，本鼠鼠还能给你再写一版“给终端用户看的超短版（1页）”和“给运营同学的处理SOP版（工单流程）”。
